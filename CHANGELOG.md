# Changelog

## 1.6.4 - 2026-04-13

### Added
- Notes desktop mode now supports exporting the current note as a `.jpg` image while keeping the existing `.txt` export entry.

### Fixed
- The desktop floating notes window now opens as an independent top-level window instead of staying attached to the main editor.

## 1.6.1修改日志

- 简化黑色小球四象限页，移除顶部说明文字，改成贴边的小图标工具栏，不再为顶部说明预留整块空白。
- 重构时间轴、文本卡片和方框卡片的界面，文本与方框都改成纯矩形卡片，去掉卡片内部按钮。
- 调整拖拽交互，时间轴、文本卡片和方框卡片都支持整块拖动，不再依赖局部拖拽把手。
- 调整方框连线方式，单击一个方框选中起点，再单击第二个方框即可创建关系箭头。
- 新增可滚动、可缩放的大画布，支持放大缩小、动态扩展绘图区，避免内容变多后放不下。
- 保留右键菜单创建文本与清空文本能力，并补充元素删除入口，便于在极简界面下继续整理画布。
- 重做四象限页 PNG 导出绘制逻辑，使导出的时间轴、文本卡片、方框卡片和连线与新界面保持一致。

## 1.5.1 - 2026-03-31

### 新增
- 新增桌面版检查更新与一键安装更新能力，设置面板内可直接检查 GitHub Releases 新版本并在下载完成后执行安装。
- 新增 GitHub Releases 自动发布工作流，推送 `v*` 标签后可自动构建并上传桌面安装包与更新清单。
- 航班模块新增“每日大赛”页面，整体沿用原有录入、保存、搜索、排序、编辑和删除逻辑，并支持按代码跳转到 `https://www.mrds66.com/archives/<代码>`。

### 优化
- 主页设置面板新增更完整的背景音乐二级菜单，支持导入外部 MP3、切换播放、停止播放和删除文件。
- 导入的音乐文件改为保存在本地浏览器数据库中，重新打开主页后会自动尝试播放上次选中的音乐。
- 优化音乐入口按钮样式，使其更接近现有设置按钮的视觉风格。
- 优化主页中心球体与周围小球的悬停逻辑，减少鼠标靠近时信息卡片发抖、卡顿的问题，并补齐中心球体的展示动画表现。
- 爱好追踪表中的打卡方格改为整格填满显示，视觉反馈更直观。
- 移除笔记系统右侧笔记编辑悬浮页中的多余按钮，简化编辑区域。

### 修复
- 修复笔记系统在点击“保存笔记”后偶发无法继续点击输入框编辑的问题，保存后会保持可继续编辑状态。
- 将笔记保存、导入等阻断式提示改为页面内提示，减少编辑流程被打断的情况。
- 修复主页音乐与设置相关状态提示不清晰的问题，补充了保存位置与下次打开自动播放的说明。

## 1.1.3 - 2026-03-30

### 新增
- 中心橙色核心球改为可点击入口，点击后沿用橙色扩散转场并打开新的 `爱好追踪表` 页面。
- 新增 `public/modules/hobby/` 月度爱好追踪模块，提供 1 到 30 日的日期列、每日事项记录列和可动态新增的爱好打卡列。
- 爱好追踪表支持新增、重命名、删除爱好列，并将每日记录与打卡状态保存到本地存储。

### 代码整理
- 新增 `scripts/check-module-consistency.mjs` 与 `scripts/module-consistency.config.json`，自动校验主页模块、桌面存储映射和 `public/modules` 目录的一致性。
- 将 Electron 词库存储映射抽离为 `electron/lib/storage-manifest.cjs`，集中管理桌面端本地存储同步配置。
- 明确 `comic` 为辅助页面模块：保留目录与页面能力，但不再作为主页注册模块参与一致性告警。
- 清理 `public/modules/words/lib/app-main.js` 中已失效的导入导出遗留分支与未使用函数，降低主控脚本噪音。

## 1.1.0 - 2026-03-28

### 主页
- 将原本右上角的桌面功能入口整合进左上角齿轮设置面板。
- 设置面板中集中提供开机自启动、导出备份、导入恢复。
- 移除主页右上角旧版 `Desktop Vault` 区块。
- 优化顶部 `SHANLIC LIFE TRACKER SYSTEM` 标题视觉，加入更强的科技感抬头样式、发光层和扫描效果。
- 主页模块入口调整，移除独立漫画小球，节省首页空间。
- 新增四象限图黑色模块小球入口。

### 笔记系统
- 保留原本完整笔记编辑流程。
- 新增页内侧边小窗编辑，可在笔记页内快速记录内容。
- 将“桌面悬浮笔记窗”入口从主页移动到笔记页面内部。
- 保留并打通独立桌面悬浮笔记窗，可在 Electron 桌面版中始终置顶使用。

### 航班与漫画
- 合并主页入口逻辑，不再单独占用漫画小球位置。
- 在航班页面保留漫画跳转按钮，点击后可直接进入漫画页面。

### 四象限图
- 新增四象限图页面与模块入口。
- 页面采用纯黑背景、中心白点和加粗十字坐标轴。
- 增加纵轴“时间”和横轴“重要”标签。
- 四个象限分别加入背景大字：`DDL`、`差事`、`想象`、`规划`。
- 支持在象限空白处右键新建纯白文本方框。
- 方框支持拖动、自由缩放、内容编辑和本地持久化保存。
- 支持长按左键“切开”方框并删除。
- 新增入场动画顺序：白点出现 -> 坐标轴延展 -> 象限背景字浮现 -> 方框弹出。
- 修复缩放后方框拖动边界计算问题。

### 单词系统
- 手动录入格式统一调整为 `单词/中文`。
- 英语与俄语录入都改为同一分隔规则，移除俄语自动判断词组逻辑。
- 更新录入框占位提示与说明文案。
- 修复词库和测试页因脚本异常导致无法进入的问题。
- 修复俄语键盘乱码，恢复正确俄语键位。
- 为俄语键盘补充 `/` 按键，方便按新录入格式输入。
- 修复词库分组卡片和删除按钮等位置的可见乱码文本。

### 文档
- 补充 `docs/release-1.1.0.md` 版本说明文档。

### 代码整理
- 抽出 `public/modules/shared/reveal.css`，统一管理多个模块页重复的进场动画样式。
- 精简 `flight`、`comic`、`lottery` 页内重复动画代码，改为共享类名驱动。
- 整理 `src/lib/modules.ts` 模块定义结构，减少重复配置样板。
- 将 `src/components/PortalScene.vue` 继续拆分为设置面板、轨道场景和动画组合逻辑，降低首页入口组件体积。
## 0.0.0 - 2026-03-27

### 优化
- 调整首页球体的渐变与阴影结构，去掉周围容易出现的黑边感。
- 优化球体放大时的发光与边缘过渡，悬停放大后不再出现明显黑色轮廓。
- 缩小笔记模块“导入 / 导出”按钮中的图标尺寸，保留原有按钮整体比例。
- 优化主页面小球点击动效：中心球体会缓慢过渡为目标颜色，被点击的小球会加速公转，中心颜色会从中心向整页扩散。
- 拉长主页面点击动画时序，按“轨道加速 -> 中心球 1 秒变色 -> 页面颜色扩散”的顺序播放。
- 进一步拆分首页点击动画阶段，延长预热和扩散时长，让三个步骤的先后顺序更容易被看清。
- 修复首页点击小球时因悬停暂停状态导致的“卡住”问题，并将点击后的公转速度调整为原来的 10 倍。
- 将首页小球调整为更干净的纯色表现，并把各模块页面主背景统一到对应球体色系，让转场后的视觉衔接更自然。
- 去掉首页球体与扩散动画里残留的深色描边和黑色渐变终点，避免放大或扩散时出现明显黑边。
- 加强抽奖转盘页左右外层背景的蓝色层次，并将装饰光斑统一为蓝色系，和主页对应球体颜色保持一致。
- 进一步加深抽奖转盘页外围蓝色背景和装饰光斑，让页面边缘的蓝色存在感更明显。
- 将“外围背景跟随小球色系”的做法同步到笔记、航班、漫画、单词模块页面，统一各页面进入后的色彩氛围。
- 优化漫画页面卡片、输入框和表格区域的层级对比，让内容区从背景中更清楚地凸显出来。
- 将漫画页面原本偏白的内容卡片统一调整为浅蓝色，并把深色展示框改成蓝色系，进一步提高同页区域对比度。
- 将单词页那种“内容依次跳出”的进场节奏扩展到笔记、航班、漫画、抽奖页面，让标题、按钮、卡片与面板在打开时分批弹出。
- 将主页颜色扩散动画的整页蒙层与中心扩散层透明度提升到最高，让点击后页面被目标颜色更彻底地覆盖。
- 将主页颜色扩散进一步改成更纯的同色染屏效果，减弱白色亮心，让整页覆盖更统一。

### 新增
- 新增桌面版开机自启动能力。
- 在首页 `Desktop Vault` 面板中加入开关按钮，可直接控制是否随系统启动。
- Windows 安装器支持自定义安装目录。
- 新增独立的桌面悬浮笔记窗。
- 悬浮笔记窗支持拖动移动、始终置顶，并可在其他软件窗口之上编辑笔记。
- 首页 `Desktop Vault` 和笔记模块内都可以直接打开悬浮笔记窗。

### 说明
- 开机自启动仅在打包安装后的桌面版中生效。
- 已安装旧版本不会自动获得这些改动，需要重新安装新安装包覆盖更新。
